📌 Project Title

 Art Collab – Collaborative Art Platform (Frontend)

📖 Project Description

Art Collab is a collaborative art platform where artists can showcase their projects, explore other creators, collaborate with them, and participate in events.

This repository contains the frontend application built using React, which connects to the backend API for authentication, project management, and user interactions.

Live Website:
🔗https://art-collab-app-mu.vercel.app/

✨ Features

🔐 User Authentication (Register / Login)

👤 User Profile Page

🎨 Create & Upload Projects

🌍 Explore Other Artists & Projects

🤝 Collaboration Option

📅 Events Page

📱 Responsive Design

🔄 API Integration with Backend

🧭 Navigation using React Router

🛠 Tech Stack Used

Frontend:

React.js

React Router DOM

JavaScript (ES6+)

CSS / Tailwind CSS 

Axios / Fetch API

Netlify (Deployment)

Backend (Connected API):

Node.js

Express.js

Supabase (Database & Auth)

⚙️ Installation Steps

Follow these steps to run the project locally:

1️⃣ Clone the Repository
git clone https://github.com/tabishachouhan/ArtCollab_App
2️⃣ Navigate to Project Folder
cd https://github.com/tabishachouhan/ArtCollab_App/tree/main/frontend
3️⃣ Install Dependencies
npm install
4️⃣ Start Development Server
npm start

The app will run on:

http://localhost:3000
